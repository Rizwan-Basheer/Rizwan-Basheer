### 🚀 About Me

I am a passionate **Full Stack Developer & Cloud Engineer** with a strong focus on building high-performance web applications, serverless architectures, and advanced web scraping/automation pipelines. I love bridging clean, modern front-end aesthetics with robust, scalable back-end engines.

- 💻 **Backend:** Specializing in Node.js (Express) and PHP (Laravel) to write secure, efficient APIs.
- 🎨 **Frontend:** Building interactive, responsive, and stunning interfaces using React.js, Tailwind CSS, and modern CSS/HTML.
- ☁️ **Cloud & DevOps:** Architecting infrastructure as code (IaC) with AWS CDK, deploying serverless solutions (AWS Lambda, DynamoDB), and automating workflows via CI/CD pipelines (GitHub Actions, GitLab CI).
- 🤖 **Automation:** Developing fast, headless automation scripts and QA testing tools using Puppeteer, Playwright, and Jest.
